## Project Installation

Clone the project

```bash
git clone https://github.com/Sonuku4564/SpendWise.git
```

Go to the project directory

```bash
cd SpendWise
```

Run the Backend (Spring Boot)

```bash
cd Backend
./mvnw spring-boot:run
```
Run the Frontend (Angular)

```bash
cd Frontend

```

Install dependencies

```bash
npm install
```

Start the server

```bash
ng serve

```
